# v1.6.5 - Exact-subject reporting insights

**Date:** 2026-07-18
**Tag:** `v1.6.5` (patch)

This release adds contract-compatible exact-subject reporting helpers to every
public Custd SDK.

## Changed

- Added closed exact-subject request types for server-owned templates, one
  pseudonymous subject, and a bounded range or date window.
- Added typed rendered-widget response contracts and malformed-response
  rejection across TypeScript, Go, Python, and PHP.
- Preserved TypeScript abort signals, Go contexts, and each SDK's existing
  authentication and transport error behavior.
- Added shared request, response, and invalid contract fixtures consumed by all
  SDK test suites.
