# v1.5.9 - Measurement proof helpers

**Date:** 2026-07-06
**Tag:** `v1.5.9` (patch)

This release adds the Go SDK helpers needed by Custd's live
`measurement-product-v1` proof generator.

## Changed

- Added Go SDK helpers for measurement calibration exercise creation and
  decision model creation under the admin measurement client.
- Added Go SDK helper for reading measurement import runs under the admin
  measurement client.
- Added Go SDK helpers for persisted forecast run create/read and decision
  simulation run create/read under `/api/v1/measurement`.
- Added Go tests that assert exact admin and analytics measurement API paths.
