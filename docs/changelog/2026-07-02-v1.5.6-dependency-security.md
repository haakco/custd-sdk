# v1.5.6 - dependency security refresh

**Date:** 2026-07-02
**Tag:** `v1.5.6` (patch)

This release clears dependency security warnings without changing SDK behavior.

## Fixed

- Removed stale root Composer lock entries that carried vulnerable
  `guzzlehttp/guzzle` and `guzzlehttp/psr7` transitive dev dependencies.
- Bumped the Go toolchain from `1.26.3` to `1.26.4` across local mise config,
  the Go SDK module, and CI workflow lint setup to pick up standard-library
  security fixes.
- Refreshed the PHP SDK lockfile content hash so `composer install` no longer
  warns about stale metadata.
