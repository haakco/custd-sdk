# v1.6.6 - Prepared-data readiness helpers

**Date:** 2026-07-19
**Tag:** `v1.6.6` (patch)

This release adds contract-compatible prepared-data readiness and canonical
output helpers to every public Custd SDK.

## Changed

- Added receipt status, output catalogue, output lookup, and canonical output
  query helpers across TypeScript, Go, Python, and PHP.
- Preserved nullable catalogue and warning states from the frozen API contract.
- Added UUID validation and cross-language request-path and response-envelope
  tests.
- Regenerated the published TypeScript distribution.
