# v1.5.8 - Measurement admin helpers

**Date:** 2026-07-05
**Tag:** `v1.5.8` (patch)

This release adds measurement product admin helpers across Go, TypeScript,
PHP, and Python SDKs.

## Changed

- Added PHP and Python helpers for measurement project create/list/get,
  observation bulk submit, single observation submit, and CSV string import.
- Added Go and TypeScript CSV import coverage for the existing measurement
  helpers.
- All measurement observation helpers validate per-row result counts and require
  successful rows to include `observationUuid`.
